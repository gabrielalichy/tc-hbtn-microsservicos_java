O que é Spring Boot Framework?
- Como o nome já diz é um framework java open source que veio para facilitar o processo em aplicações Java.

Quais são os benefícios e recursos que Spring Boot fornece?
- Ofere uma experiência de inicio de projeto rápida.
- É flexivel sobre como os projetos Spring são configurados.
- Utiliza o pom.xml para a organização de dependências (Maven), de forma fácil e organizada.

Como é feita a adição de novas dependências (JPA, Hibernate, etc…)?
- Feita através do arquivo POM.XML

Na sua opinião, qual é a melhor IDE para trabalhar com o Spring Boot: Spring Tools Suites, IntelliJ IDEA, Eclipse ou NetBeans?
- Eu gosto do Eclipse por sua variedade de plug-ins pra personalizar meu ambiente de acordo com a necessidade do meu projeto.

O que é Spring Initializr?
- O Spring Initializr fornece uma interface web mais simples para o usuário, gerando seu projeto a partir de um estrutura de configs pré-moldadas.
